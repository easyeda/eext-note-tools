## 使用方法
1.在"设置"-"扩展"-"扩展管理器"中导入docs-tools.eext扩展文件。
![63de0379903d4efa94bf9d8297762fb5](https://github.com/user-attachments/assets/9bc4eeed-a9ae-4400-ba88-051b7fd4e020)


2.进入PCB界面，点击顶部导航栏"DocsTools"选择需要的功能即可。
